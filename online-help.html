Online Help

Project Summary

This documentation was created as a term project for COMM 1260: Design & Write Online Help, taken as part of the Technical Writing Certificate at the British Columbia Institute of Technology.

The goal of the project was to use MadCap Flare to create a functioning help system with five fully-written, search engine optimized topics, supported by 15 "blank" topics and an organized table of contents.
