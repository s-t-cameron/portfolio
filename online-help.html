HTML5 Online Help

This documentation was created as a term project for COMM 1260: Design & Write Online Help, taken as part of the Technical Writing Certificate at the British Columbia Institute of Technology. The goal of the project was to use MadCap Flare to create a functioning help system with five fully-written, search engine optimized topics, supported by 15 "blank" topics and an organized table of contents.

Documentation Highlights

While the entire help file is available to explore, many of the topics are just placeholders. This section goes over some of the finished content, linking directly to the pages when relevant.

The Welcome Page

The marketing of Ingress often blurs the line between reality and fiction. Its promotional material frequently leans into intrigue around the lore. For consistency with that branding, the first impression of the site is partially "in-universe"; it plays into that sense of mystery while still delivering enough information to be helpful.

For further brand alignment, the colour pallette for the help file is based on the app itself.

SEO

Each topic has been carefully written to maximise the effectiveness of search. As an example, almost every in-game action earns progress toward a specific medal. The addition of metadata ensures that if a user searches for the name of a medal, the topic for the relevant in-game action appears first to allow them to learn more about it.
